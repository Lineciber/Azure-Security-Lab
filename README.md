# 🛡️ Azure Security Lab (AZ-500)

Bem-vindo ao meu laboratório de segurança em nuvem! Este repositório documenta minha jornada de estudos para a certificação **Microsoft AZ-500** e a aplicação prática desses conceitos no dia a dia de um **SOC**.

## 🚀 Objetivos
- Implementar controles de identidade (Entra ID).
- Configurar segurança de rede (NSG, Azure Firewall).
- Gerenciar proteção de dados (Key Vault, Storage Security).
- Monitoramento e resposta a incidentes com Microsoft Sentinel.

## 📁 Estrutura do Repositório
* **/Identity:** Gestão de usuários, MFA, PIM e Acesso Condicional.
* **/Networking:** Configurações de VNETs, NSGs e Firewall.
* **/Data-Protection:** Backup, Encryption e Key Vault.
* **/Sentinel-SOC:** Consultas KQL e automações de resposta.
